Projekt: Bildbetrachter, Version 1.0
Autoren: David Barnes und Michael Kölling

Dieses Projekt ist Teil des Zusatzmaterials zum Buch

   Java lernen mit BlueJ.
   Objects first - eine Einführung in Java, 6. Auflage
   David J. Barnes und Michael Kölling
   Pearson Education Deutschland, 2017
Es wird in Kapitel 13 besprochen.

Zur Benutzung dieses Projekts erzeugen Sie eine Instanz der
Klasse Bildbetrachter. Dies ist die erste vollständige Version dieses Projektes. Sie ist noch nicht perfekt, aber zumindest kann sie bereits Bilder anzeigen!

Weitere Details können Sie im Buchkapitel nachlesen.
