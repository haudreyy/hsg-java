Projekt: Bildbetrachter-komplett (Version 3.1)
Autoren: David Barnes und Michael Kölling

Dieses Projekt ist Teil des Zusatzmaterials zum Buch

   Java lernen mit BlueJ.
   Objects first - eine Einführung in Java, 6. Auflage
   David J. Barnes und Michael Kölling
   Pearson Education Deutschland, 2017
Es wird in Kapitel 13 besprochen.

Dieses Projekt demonstriert die Konstruktion einer einfachen grafischen Benutzungsoberfläche (mit Labels, Buttons, Layout-Managern, Ereignisverarbeitung, Menüs, Dialogen) sowie ein wenig Bildverarbeitung.

Sie starten dieses Projekt, indem Sie ein Objekt der Klasse Bildbetrachter erzeugen. Dies lässt automatisch die GUI der Anwendung erscheinen - es ist keine weitere Interaktion mit dem Objekt auf der Objektleiste notwendig.

ACHTUNG:

Es passiert leider häufig, dass das Fenster des Bildbetrachters HINTER dem BlueJ-Fenster erscheint. Das kann etwas verwirren, weil es so wirkt, als würde nichts passieren.
