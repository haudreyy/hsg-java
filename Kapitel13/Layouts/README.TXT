Projekt: Layouts
Autoren: David Barnes und Michael Kölling
Version: 2016.02.29

Dieses Projekt ist Teil des Zusatzmaterials zum Buch

   Java lernen mit BlueJ.
   Objects first - eine Einführung in Java, 6. Auflage
   David J. Barnes und Michael Kölling
   Pearson Education Deutschland, 2017

Dieses Projekt demonstriert verschiedene Layout-Optionen.

Benutzungshinweis: Erzeugen Sie ein Objekt einer der Beispielklassen. Verändern Sie die Fenstergröße und beobachten Sie, wie sich dies auf Größe und Anordnung der Komponenten auswirkt.
