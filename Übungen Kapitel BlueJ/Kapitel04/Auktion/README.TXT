Projekt: Auktion
Autoren: David Barnes und Michael Kölling

Dieses Projekt ist Teil des Zusatzmaterials zum Buch

   Java lernen mit BlueJ.
   Objects first - eine Einführung in Java, 6. Auflage
   David J. Barnes und Michael Kölling
   Pearson Education Deutschland, 2017

Es wird in Kapitel 4 besprochen.

Zweck des Projektes: Sammlungen von Objekten demonstrieren

Startpunkt:
    Erzeugen Sie ein Exemplar der Klasse Auktion.
    Melden sie einige Auktionsposten mit der Operation
    'postenAnmelden' an (ein Posten bei einer Auktion
    kann ein einzelner Gegenstand oder eine Zusammenstellung
    von Gegenständen sein, die versteigert werden sollen).
    Es wird lediglich eine Zeichenkette als Beschreibung
    des Postens benötigt.
    Erzeugen Sie eine oder mehrere Person-Objekte, die
    die Bieter repräsentieren sollen.
    Lassen Sie sich die Posten der Auktion anzeigen und
    wählen Sie eine Postennummer, für die geboten werden soll.
    Lassen Sie sich den Posten auf die Exemplarleiste legen.
    Geben Sie ein Gebot, indem Sie die Methode gibGebotAb für
    das Auktion-Objekt aufrufen. Übergeben Sie die Postennummer,
	die bietende Person und die Höhe des Gebots.
