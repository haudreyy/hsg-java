Projekt: Baelle 
Autoren: Michael Kölling und David J. Barnes

Dieses Projekt ist Teil des Zusatzmaterials zum Buch

   Java lernen mit BlueJ.
   Objects first - eine Einführung in Java, 6. Auflage
   David J. Barnes und Michael Kölling
   Pearson Education Deutschland, 2017

Es wird in Kapitel 6 besprochen.

Ziel: Demonstration der grafischen Möglichkeiten auf einer Canvas-Klasse.

Dieses Projekt bietet eine kurze grafische Demonstration in der
Klasse BallDemo. 

Dieses Projekt bietet Raum für Erweiterungen.
Die Klasse Canvas braucht üblicherweise nicht geändert zu werden.
Sie sollte eher wie ein Bibliotheksklasse behandelt werden (öffnen Sie den Editor und wechseln Sie in die Schnittstellensicht). 

Die anderen Klassen können geändert, erweitert oder um weitere Klassen ergänzt werden. Das Buch gibt Anregungen für verschiedene Übungen.
