Projekt: Adressbuch-V2T
Autoren: David J. Barnes und Michael Kölling

Dieses Projekt ist Teil des Zusatzmaterials zum Buch

   Java lernen mit BlueJ.
   Objects first - eine Einführung in Java, 6. Auflage
   David J. Barnes und Michael Kölling
   Pearson Education Deutschland, 2017

Es wird in Kapitel 14 besprochen.

Eine einfache Implementierung eines Adressbuches.

Startpunkt: 
    Entweder: Sie erzeugen ein Objekt der Klasse Adressbuch
    und fügen Kontaktdaten (wie Name, Adresse, Telefon) ein.
    Oder: Sie erzeugen eine AdressbuchDemo und rufen ihre
    Methode zeigeSchnittstelle auf.
    Dies wird ein Adressbuch mit Testdaten erzeugen und eine
    Textschnittstelle, mit der interaktiv das Adressbuch
    bearbeitet werden kann.
