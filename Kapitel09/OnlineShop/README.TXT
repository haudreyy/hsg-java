Projekt: Online-Shop
Autoren: Michael Kölling und David J. Barnes

Dieses Projekt ist Teil des Zusatzmaterials zum Buch

Java lernen mit BlueJ.
   Objects first - eine Einführung in Java, 6. Auflage
   David J. Barnes und Michael Kölling
   Pearson Education Deutschland, 2017
Dieses Projekt implementiert einen kleinen Teil eines Onlineshop-Systems (wie z.B. Amazon.com). Das aktuelle Projekt konzentriert sich ganz auf die Kommentare, die die Kunden für die Produkte abgeben. Es enthält Code zum Erzeugen, Speichern, Anzeigen
und Bearbeiten von Kundenkommentaren.

HINWEIS: In diesen Klassen sind verschiedene ernste Fehler enthalten.
