Projekt: Ziegelsteine
Autoren: Michael Kölling und David J. Barnes

Dieses Projekt ist Teil des Zusatzmaterials zum Buch

   Java lernen mit BlueJ.
   Objects first - eine Einführung in Java, 6. Auflage
   David J. Barnes und Michael Kölling
   Pearson Education Deutschland, 2017

Dies ist Teil einer Anwendung in einer Ziegelfabrik.
Ziegel werden in Paletten ausgeliefert (Stapel von Ziegeln).
    
Eine Palette soll Methoden anbieten, die ihre Höhe und ihr
Gewicht liefern.

Es gibt (mindestens) vier Fehler in diesem Projekt. Finden Sie
diese. Beheben Sie sie. Diskutieren Sie dann, wie Sie bei der
Suche vorgegangen sind. Welche Strategien haben Sie angewendet?
