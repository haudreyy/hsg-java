Projekt: Taxi-spaetere-Stufe
Autoren: David J. Barnes und Michael Kölling

Dieses Projekt ist Teil des Zusatzmaterials zum Buch

   Java lernen mit BlueJ.
   Objects first - eine Einführung in Java, 6. Auflage
   David J. Barnes und Michael Kölling
   Pearson Education Deutschland, 2017
Es wird in Kapitel 16 besprochen.

Dieses Projekt bietet eine Teilimplementierung einer Simulation von Taxis, die innerhalb einer Stadt Fahrgäste aufnehmen und transportieren.

Dies ist die vierte Version dieses Projekts. Sie zeigt die Implementierung des Betriebs mit mehreren Taxis.

Startpunkt:
    Erzeugen Sie ein Objekt der Klasse Simulation und rufen Sie seine Methode starten auf.

Benutzen Sie diese Version des Projekts, um weitere Ideen umzusetzen, beispielsweise die Implementierung der Klasse Grossraumtaxi.
