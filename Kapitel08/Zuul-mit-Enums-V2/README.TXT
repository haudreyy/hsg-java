Projekt: Zuul-mit-Enums-V2
Autoren: Michael Kölling und David J. Barnes

Dieses Projekt ist Teil des Zusatzmaterials zum Buch

   Java lernen mit BlueJ.
   Objects first - eine Einführung in Java, 6. Auflage
   David J. Barnes und Michael Kölling
   Pearson Education Deutschland, 2017

Dieses Projekt ist das Grundgerüst für ein Adventure-Game.
In dieser Version hat es ein paar Räume und bietet dem Spieler die Möglichkeit, sich zwischen diesen Räumen zu bewegen. Das ist alles.

Um ein Spiel zu starten, erzeugen Sie eine Instanz von 'Spiel' und rufen die Methode 'spielen' auf.

Dieses Projekt wurde als Startpunkt für ein kleines Java-Projekt
entworfen.

Dieses Projekt wurde geschrieben, um die Verwendung von Aufzählungstypen zu illustrieren, wodurch die Sprachenunabhängigkeit der Spiellogik unterstützt wird.
  
Lesen Sie Kapitel 8 für eine detaillierte Beschreibung dieses Projekts.
